# Operations Scripts

This folder contains the small utilities that sit around the research repo
instead of the simulation pipelines themselves.

## Contents

- [Run thermal tmux session](run_thermal_tmux.sh)
- [Submit gem5 Gerrit change](submit_gerrit.sh)
- [Submit gem5 GitHub issue](submit_issue.py)

## Use this folder when you need

- To launch the thermal pipeline in a tmux session
- To prepare and push the thermal fix to Gerrit
- To submit the issue report to GitHub from the local write-up
