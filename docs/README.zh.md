## 建議閱讀順序

1. 先看主入口：
   - [英文版](../README.en.md)
   - [中文版](../README.zh.md)
2. 再讀 [方法論](methodology.md)，理解熱模型、求解器與 gem5 設定。
3. 接著看 [驗證結果](validation.md)，確認交叉驗證與數值證據。
4. 再讀 [模型限制](limitations.md)，了解假設與邊界。
5. 看 [相關研究](related-work.md)，把本專案放回 HotSpot、McPAT 與 SPICE 的脈絡。
6. 最後閱讀 [真實硬體溫度校準指南](real_device_calibration.zh.md) 了解如何測量真實物理參數以反饋至模擬。
